The primary dataset (insurance.csv) contains comprehensive information about health insurance customers with 7 key variables:

Initial Data Quality Assessment

The initial exploration revealed several data quality issues that required comprehensive cleaning before analysis, including negative ages, inconsistent categorical values, and formatting issues in the charges column.

Data Cleaning Process

Issue 1: Negative Ages

Problem: Found negative age values (e.g., -31, -27, -23)

Solution: Converted negative ages to positive by taking absolute values

Issue 2: Inconsistent Gender Values

Problem: Multiple representations (female, woman, F, male)

Solution: Standardized to 'male' and 'female' only

Issue 3: Inconsistent Region Capitalization

Problem: Mixed case regions (Southwest, southwest, Southeast, etc.)

Solution: Standardized all regions to lowercase

Issue 4: Charges Formatting

Problem: Charges column contained '$' symbols

Solution: Removed currency symbols and converted to numeric

Additional Cleaning Steps

  • ‚Ä¢ Checked for and handled missing values
  • ‚Ä¢ Removed duplicate records
  • ‚Ä¢ Validated data types and ranges
  • ‚Ä¢ Ensured data consistency across all columns
